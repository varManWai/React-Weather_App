# Description
This website is a __*Weather Checker*__ which allow the user to __enter the city or country__ to get the __Temperature and Weather currently__.

# This is created by 
  * __HTML__
  * __CSS__ 
  * __React JS__
  * __Vite JS__
  * __OpenWeatherAPI__
